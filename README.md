В данном проекте я хотел реализовать идею сайта продажи скинов (Как пример использовал сайт [market.csgo](https://market.csgo.com/ru/)).

Для реализации этого проекта я использовал: Spring core, Spring Data, Spring Security, PostgreSql, шаблонизатор Freemarker, базовый css и немного js для ajax.

Архитектура проекта - MVC.

Это проект даже вряд-ли находится на стадии 'MVP', так как он был больше как учебный и делался по требованиям к семестровой работе. Поэтому я хочу использовать эту же идею, но реализовать её в другом проекте - https://github.com/imonr65/skins-marketplace-2.0
